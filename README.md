# RequireAI - BRD Intelligence Platform

Enterprise-grade requirements analysis and BRD generation powered by Gemini.

## Features
- 9-phase AI pipeline
- Interactive Knowledge Graph
- AI Copilot
- Conflict Detection
- Multi-format Export (PDF, Markdown, DOCX)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
